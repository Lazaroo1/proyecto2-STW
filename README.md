# Calculadora STW

A React calculator built with TypeScript and Bun. The app includes calculator state
management, accessible controls, component tests, Storybook stories, and a Vite
development setup.

## Installation

```bash
bun install
```

## Run the App

```bash
bun dev
```

## Run Tests

```bash
bun vitest run
```

You can also use the project test script:

```bash
bun test
```

## Run Lint

```bash
bun run lint
```

## Run Storybook

```bash
bun storybook
```
